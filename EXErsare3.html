Text link
  1. href este atributul în care se adaugă adresa link-ului.
  2. url este adresa link-ului (a paginii care va fi deschisă).
  3. Titlu link este un text ascuns care apare când este poziţionat mouse-ul deasupra link-ului.
  4. Text link este textul care va aparea în browser şi pe care trebuie dat click cu mouse-ul.

In caz in care link-ul este spre o pagina spre o pagina din acelasi repositoriu atunci se poate folosit doar numele de file, in loc de tot traseul
